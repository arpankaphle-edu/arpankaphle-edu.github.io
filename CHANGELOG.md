# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-01-19

### Added
- **SEO Enhancements** - Full search engine optimization across all pages
  - Open Graph meta tags for social sharing (Facebook, LinkedIn)
  - Twitter Card meta tags
  - Canonical URLs for each page
  - Keywords and author meta tags
  - `sitemap.xml` for search engine crawling
  - `robots.txt` with sitemap reference
  - JSON-LD structured data (Person schema) on home page
- LinkedIn button added to hero section
- Expanded footer across all pages with LinkedIn, Fun Site, and GitHub links
- Note about private GitHub repos on both `arpankaphle-edu` and `break-through` accounts
- MIT UAV Club link in projects page footer

## [1.0.0] - 2026-01-19

### Added
- **Initial Release** - Professional personal website for GitHub Pages

#### Pages
- `index.html` - Home/About page with professional bio and quick links
- `experience.html` - Work experience timeline (Google, MIT/Cadence, Amazon, Brain Power, Buddi)
- `education.html` - Degrees and selected MIT coursework organized by category
- `projects.html` - Portfolio of fun undertakings and hobbies

#### Features
- Clean, minimal design with professional typography (Inter font)
- Responsive navigation with mobile hamburger menu
- CSS custom properties for easy theming
- Subtle animations and hover effects
- Accessible markup with proper ARIA labels
- SEO-friendly meta descriptions

#### Technical
- `assets/css/styles.css` - Complete design system with tokens
- `assets/js/main.js` - Navigation and page interaction logic
